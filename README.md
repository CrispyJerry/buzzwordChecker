The following program is a simple program that checks for the most common buzzwords that recruiters are tired of seeing in someones Resume or CV.

pdfToText() converts pdf to text

buzzwordChecker checks for words using Regex

BUZZWORDS is .txt file with buzzwords which can be modified easily to add your own words if you wish.

PDF-Path is your file location for your PDF.



All code works as of this moment and plans to add the following are in the works:

UI
Support for different filetypes such as .DOC .DOCX etc...
Resume/CV score
Count of each buzzword located
